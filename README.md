# comunicados-bcb-automaticos
O presente codigo tem o objetivo de extrair os comunicados do BCB de forma automatica e rapida por meio do API do banco central, dado que os comunicados anteriores ao 45 não estão disponiveis a coleta começa no comunicado 46.
